connecting-line2: screen pick — px raycast (orthographic camera)

Move the mouse over each line. This is simple-draw's 2D case. The cursor carries raycaster-thresholds="line2Mode:set; line2:8", so params.Line2.threshold = 8px (px because these targets detect in screen space — matchLine's metres would be unpickable here, which is exactly why px targets use line2Mode:set).

  • cyan — raycastUnits:px: stock screen-space raycast, ~8px grab band around the stroke.
  • red — raycastUnits:auto: resolves to px under this orthographic camera — should behave identically to cyan.

Hover tints the line yellow and updates the status line below. Both should pick from a few px away from the 1px stroke (the 8px band), matching stock Line2 screen-space behaviour.
