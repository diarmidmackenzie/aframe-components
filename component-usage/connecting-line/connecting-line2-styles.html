connecting-line2: static pattern gallery.

Each row is a different dash pattern, shown at two widths (left = 6px, right = 3px). Dash values are world metres (this is a perspective scene, so dashUnits:auto resolves to world units), sized for the ~2 m lines. The "dotted" entry is butt-capped squares (LineMaterial has no round caps), not round dots.
